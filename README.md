## Бізнес-проблема
**Утримання клієнтів (Retention)** коштує в 5-10 разів дешевше, ніж залучення нових. У цьому проєкті я досліджую дані про поведінку користувачів сервісу, щоб виявити "червоні прапорці" — ознаки того, що клієнт збирається піти, та надати бізнесу конкретні рекомендації щодо їх утримання.
### 🛠 Технічний стек
* **Інструменти:** Python (Pandas для обробки, Seaborn/Matplotlib для візуалізації, Scikit-learn — логістична регресія).
* **Методологія:** RFM-аналіз (Recency, Frequency, Monetary), кореляційний аналіз, когортний аналіз.
* **Дані:** Датасет з інформацією про транзакції, тип контракту та активність клієнтів.
### 💡 Етапи роботи
1. **Data Audit та очищення**
* **видалення null CustomerID**
* **фільтрація повернень і нульових цін**
* **створення Revenue**
2. **Customer-level агрегація**
* **Recency, Frequency, Monetary**
* **Lifetime, Avg order value**
3. **Churn definition**
* **churn = відсутність покупок > 180 днів**
4. **RFM-сегментація**
* **виділення Champions, At Risk, Lost тощо**
5. **Retention / Cohort analysis**
* **аналіз повернення клієнтів у часі**
6. **Correlation & Predictive modeling**
* **визначення драйверів churn**
* **побудова логістичної моделі**
### 🎯 Виклики
* **Велика кількість транзакцій без CustomerID (~25%)**
* **Наявність повернень і від’ємних значень**
* **Багато одноразових покупців (складність прогнозу)**
* **Дисбаланс класів churn / active**
### 📈 Головні інсайти та рекомендації
* **Інсайт 1:** Найбільший відтік відбувається після першої покупки.
* **Інсайт 2:** Ключовий фактор churn — зростання recency_days.
* **Інсайт 3:** Частота покупок має сильний захисний ефект.
* **Інсайт 4:** Сума чека менш впливає на відтік, ніж регулярність.
* **Інсайт 5:** Retention різко падає у перші 1–2 місяці.
### 💡 Рекомендації
1. Фокус на повторній покупці в перші 30–60 днів
2. Автоматичні тригерні кампанії при зростанні recency
3. Програми лояльності для підвищення частоти
4. Використання churn probability для пріоритезації клієнтів
5. Окремі стратегії для сегментів “At Risk” і “New Customers”
